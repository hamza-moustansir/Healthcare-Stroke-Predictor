# Healthcare Stroke Predictor

Une application React pour prédire le risque de stroke et fournir des recommandations médicales personnalisées.

## Fonctionnalités
- Prédiction du risque de stroke en fonction de l'âge, de l'hypertension, des maladies cardiaques, du niveau de glucose et de l'IMC.
- Affichage des recommandations médicales dans une pop-up stylisée.

## Technologies utilisées
- Flusk
- Q-learning
- React
- Tailwind CSS
- SweetAlert2 (pour les pop-ups)
- Axios (pour les requêtes API)
